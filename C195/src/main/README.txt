# Application Title and Purpose

## Title
Customer Appointment Management System

## Purpose
The Customer Appointment Management System is designed to help manage customer appointments efficiently. It allows users to log in, view, add, update, and delete customer appointments, and generate various reports regarding customer interactions.

# Instructions to Run the Program

1. **Clone the repository or download the source code** to your local machine.
2. **Open the project in IntelliJ IDEA** (or your preferred IDE).
3. **Ensure you have the correct JDK and JavaFX SDK** set up in your project structure.
4. **Run the `Main.java` file** located in `src/main/java/com/example/c195/`.
5. **Log in using your credentials** (or create a new account if registration is available).
6. **Navigate through the application** to manage customer appointments and generate reports.

---

# Description of Additional Report

The additional report provided in the application is a detailed summary of customer appointments by type and month. This report helps in understanding the distribution of different types of appointments over time and assists in resource planning and management.
